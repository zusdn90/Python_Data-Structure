## Queue
- Enqueue: queue에 데이터를 넣는 기능
- Dequeue: queue에 데이터를 꺼내는 기능
- Python에서는 Queue(), LifoQueue(), PriorityQueue()를 제공
    + Queue(): 가장 일반적인 큐
    + LifoQueue(): 나중에 입력된 데이터가 먼저 출력(스택구조)
    + PriorityQueue(): 데이터마다 우선순위를 넣어서, 우선순위가 높은 순으로 데이터 출력
